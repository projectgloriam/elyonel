# Elyonel Fruits

This is a Rails web app used for Elyonel Fruits POS.

Things you may need:

* Ruby version 2.3.3

* Rails version 5.2.2

## How to get started

Clone the repository:
```
$ git clone https://github.com/projectgloriam/elyonel.git 
```

Change to the directory
```
$ cd elyonel
```

Install the gems
```
$ bundle install 
```

Migrate the database:
```
$ rails db:migrate
```

Run the app
```
$ rails s
```
